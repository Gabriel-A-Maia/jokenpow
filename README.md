# 🪨📄✂️ JokenPô Game

Um jogo clássico de Pedra, Papel e Tesoura desenvolvido com JavaScript puro, focado em prática de lógica, manipulação do DOM e interatividade.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎮 Funcionalidades

* Escolha entre pedra, papel ou tesoura
* Jogada aleatória da máquina
* Exibição do resultado em tempo real
* Contador de pontuação (jogador vs máquina)
* Interface simples e responsiva

---

## 🧠 Aprendizados

Neste projeto, foram aplicados conceitos importantes como:

* Manipulação do DOM
* Eventos (`addEventListener` e `onclick`)
* Estruturas condicionais (`if/else`)
* Funções em JavaScript
* Atualização dinâmica de conteúdo na tela
